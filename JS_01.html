變數與資料型別

打開 F12 → Console，觀察以下程式輸出：


var name = "小明";
var age = 18;
var isStudent = true;

console.log(name);            // 小明
console.log(typeof name);     // string

console.log(age);             // 18
console.log(typeof age);      // number

console.log(isStudent);       // true
console.log(typeof isStudent);// boolean
